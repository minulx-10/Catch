# 캐치봇 / 생물 포획 RPG 기획서

## 1. 게임 개요
- **게임명**: 캐치봇 (가칭) / 생물 포획 RPG
- **핵심 목표**: 지구상의 다양한 생물을 탐색, 포획, 수집하는 RPG.
- **주요 요소**: 탐색, 포획, 도감, 랭킹
- **방향성 합의 필요**: '힐링 채집겜' vs '하드코어 수집형 배틀 RPG'. 현재는 1차적으로 탐색+포획+도감+랭킹 중심의 수집 게임으로 개발 시작 권장.

## 2. 1차 버전 목표 (포획 중심)
- `./탐색` 명령어 사용
- 랜덤 바이옴 3개 출력
- 바이옴 선택
- 생물 랜덤 등장
- 포획 성공/실패
- 도감 등록
- 생물 등급 표시
- 간단한 랭킹

## 3. 바이옴 및 날씨 시스템
- **바이옴**: 숲, 들판, 툰드라 (추후 바다, 사막, 동굴 등 확장 예정)
- **날씨**: 맑음, 비, 흐림, 악천후, 비 온 뒤 흐림. 날씨에 따라 특정 생물의 등장 확률이 변화함.

## 4. 등급 및 속성
- **등급**: N(노말) -> R(레어) -> SR(슈퍼 레어) -> UR(울트라 레어) -> LR(레전드 레어)
- **속성**: 군집형, 돌격형, 자기 보호형, 민첩형, 초연함 (전투 역할/행동 패턴 기준)
- **위험도**: 하급(1~100), 일반(101~300), 강성(301~700), 포식(701~1500), 재앙(1501~)

## 5. 초기 생물 데이터 (N등급)
- **벼메뚜기**: 전투력 5, 군집형, 숲/들판. (비 오면 확률 감소)
- **개미**: 전투력 1, 군집형, 숲/들판. (비 오면 확률 감소)
- **지렁이**: 전투력 3, 자기 보호형, 숲/들판. (비 오거나 비 온 뒤 등장)
- **들개**: 전투력 100, 돌격형, 숲/들판. (날씨 안 좋으면 확률 감소)
- **배추흰나비**: 전투력 1, 관상형/회피형, 숲/들판. (날씨 안 좋으면 확률 감소)

---
*참고: 배틀 시스템(유저 PVP, 콜로세움 등)은 1차 버전에서는 제외 혹은 매우 단순화하며, 2~3차 업데이트로 미룸.*
